# 📃 프로젝트 소개

AGROUNDS는 아마추어 축구 커뮤니티의 요구를 반영하여 기획된 서비스로, 경기 기록 및 전술 분석을 통해 팀과 선수들이 더 나은 경기를 할 수 있도록 돕습니다. 이 데모 버전은 일부 유저들에게 배포되어 피드백을 수집하고, 이를 바탕으로 필요한 기능을 강화하고 불필요한 기능은 제거하여 서비스의 완성도를 높일 계획입니다.
